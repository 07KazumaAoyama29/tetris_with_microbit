# micro:bitでブロック崩しゲーム草案
## 概要
- micro:bitでブロック崩しゲームについての資料です
- resoucesフォルダにテキストを置いています
- programフォルダにテンプレートと見本プログラムを置いています。

変数・if文・while文・条件式について書いておく。<br>
